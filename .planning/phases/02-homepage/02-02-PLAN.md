**src/components/ui/Button.tsx**

Props interface:
```typescript
interface ButtonProps {
  variant?: 'primary' | 'secondary' | 'ghost';
  size?: 'sm' | 'md';
  href?: string;           // if provided, renders as Next.js Link; otherwise renders as <button>
  className?: string;
  children: React.ReactNode;
  type?: 'button' | 'submit' | 'reset';
  disabled?: boolean;
  onClick?: () => void;
}
```

Variant styles (use exact Tailwind token classes — NO hardcoded hex values):
- `primary`: `bg-gold text-ink hover:bg-gold-hover transition-colors duration-200`
- `secondary`: `border border-gold text-gold bg-transparent hover:bg-gold/10 transition-colors duration-200`
- `ghost`: `text-silver hover:text-mist border border-slate/40 hover:border-slate/70 transition-colors duration-200`

Size styles:
- `sm`: `px-4 py-2 text-small`
- `md` (default): `px-6 py-3 text-sm`

Shared styles applied to all variants: `inline-flex items-center justify-center font-medium rounded-[var(--radius-button)] disabled:opacity-50 disabled:cursor-not-allowed`

Logic: if `href` is provided, import `Link` from `next/link` and render `<Link href={href} className={...}>{children}</Link>`. Otherwise render `<button type={type ?? 'button'} disabled={disabled} onClick={onClick} className={...}>{children}</button>`.
